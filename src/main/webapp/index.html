Metadata Transformation Engine

This is a simple, first pass at a server-hosted metadata transformation engine.

To run a transformation, use the following form to upload:

  1. An XSLT file
  2. A Zip file (with the XML metadata records to be transformed)

At this time, only Zip files are supported for metadata upload. The stylesheet can use XSLT 2.0.

Rules: Metadata file names, within the Zip file, must end with a ".xml" extension.
